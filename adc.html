ADC - ArchivedDataCodec

Version 1.1.0

Introduction

ADC (ArchivedDataCodec) is a new file extension and file archiver that uses zlib for efficient compression and decompression of various file types. It features a command-line interface, supporting both Windows and Linux.

Features

  • Efficient zlib-based compression.
  • Cross-platform compatibility (Windows & Linux).
  • Simple command-line interface for ease of use.

Installation

Windows

Download the latest release from the GitHub releases page.

Linux

  1. Install Git: sudo apt install git
  2. Clone the repository: 
    git clone https://github.com/Mealman1551/ADC.git
cd ADC/Linux-release
chmod +x adc1.1
./adc1.1

Usage

To archive or extract files, use the following commands:

./adc1.1 --compress myfile.txt --output myfile.adc

For decompression:

./adc1.1 --extract myfile.adc --output myfile.txt
