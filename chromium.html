ChromiumUp

The Chromium Updater or also known as 'chromium-latest-Enhanced' is a fork of scheib/chromium-latest-linux by Mealman1551 to make installing and updating Chromium way easier

Click on the logo or following link to go to the repository

Features

  • Downloads and installs the latest stable version of Chromium for
  • AMD64/x64 by typing 1 command
  • Lightweight script with minimal dependencies
  • Automatically creates a desktop entry for easy access
  • Supports multiple languages

How does it work

The repo contains these files:

What does each file do?

  • chromiumup: This file is the main updater for Linux, this is a simple Bash script that can be invoked with: chromiumup
  • install-windows.ps1: This script is for installation on Windows, you need to download the script locally on Windows 10 and for Windows 11 and up you can use the irm command.
  • install.sh: This is the script for installation on Linux.
  • LICENSE: The license of this project
  • README.md: Documentation for the project
  • remove.sh: This script will remove Chromium and chromiumup from Linux.
  • uninstall-windows.ps1: This script will remove Chromium and chromiumup from Windows

Notes

  • This updater only supports AMD64/x64 architecture
  • This script pulls the "raw" build, so no proprietary codecs, no Widevine and no Google API keys

Installing, updating and removing Chromium

Linux

To install and update Chromium on Linux, follow these steps:

  1. Open your terminal.
  2. Run the following command to download and install the updater script: 
    wget -qO- https://raw.githubusercontent.com/Mealman1551/chromium-latest-Enhanced/refs/heads/master/install.sh | bash
  3. Once installed, you can update Chromium at any time by running: 
    chromiumup

To remove Chromium on Linux, follow these steps:

  1. Open your terminal.
  2. Run the following command to uninstall Chromium: 
    wget -qO- https://raw.githubusercontent.com/Mealman1551/chromium-latest-Enhanced/refs/heads/master/remove.sh | bash

To completely remove Chromium on Linux use this: (Not recommended)

wget -qO- https://raw.githubusercontent.com/Mealman1551/chromium-latest-Enhanced/refs/heads/master/remove_complete.sh | bash

Windows

For Windows 10 you need to download the script locally/manually. Windows 11 and up work with the irm command.

To install and update Chromium on Windows, follow these steps:

  1. Open PowerShell as Administrator.
  2. Run the following command to download and install the updater script: 
    irm https://raw.githubusercontent.com/Mealman1551/chromium-latest-Enhanced/refs/heads/master/install-windows.ps1 | iex
  3. Once installed, you can update Chromium at any time by running: 
    chromiumup

To remove Chromium on Windows, follow these steps:

  1. Open PowerShell as Administrator.
  2. Run the following command to uninstall Chromium: 
    irm https://raw.githubusercontent.com/Mealman1551/chromium-latest-Enhanced/refs/heads/master/uninstall-windows.ps1 | iex

Important Warning for Users

This script installs a direct raw snapshot Chromium build alongside any existing Chromium installation.

  • It may overwrite or conflict with your current Chromium profile.
  • Possible issues include:
    • Chromium failing to start due to a corrupt profile.
    • Loss of settings, extensions, or saved passwords if using the same profile.

To troubleshoot, launch Chromium with a temporary clean profile:

chromium --user-data-dir=/tmp/chromium-test

Or back up your current profile:

mv ~/.config/chromium ~/.config/chromium.backup

Use this script only if you intend to test a separate build and are willing to back up your profile.
